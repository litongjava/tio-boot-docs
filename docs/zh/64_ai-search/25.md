# Docker 部署

```java
docker run -dit --net=host --name max-search-web \
-e JDBC_URL=jdbc:postgresql://192.168.3.9/max_kb \
-e JDBC_PSWD= \
-e TAVILY_API_TOKEN= \
-e VOLCENGINE_API_KEY= \
-e GEMINI_API_KEY= \
litongjava/max-search-web:1.0.0

```

## 安装 postgresql

https://www.tio-boot.com/zh/41_postgresql/01.html

## 初始化数据

开启向量
-- 切换到 max_kb 数据库

```
\c max_kb
CREATE EXTENSION IF NOT EXISTS vector;
```

导入数据 db\init.sql

我已经安装了新的 postgresql 数据库,名称是名是 pgmaster,如何在启动新容器时设置环境变量为
JDBC_URL=jdbc:postgresql://$(pgmaster)/max_kb
