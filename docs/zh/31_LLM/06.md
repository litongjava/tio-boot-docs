# 结构化数据检索

## 什么结构化数据检索

结构化数据检索 和 文档检索(非结构化数据) 的检索思路相同.不同的是 这部分使用的是 text-to-sql.经用户的自然语言(text)转为数据库查询语言(sql)

## 结构化数据检索 流程

- 1.数据:一般是采用爬虫的形式从互联网上爬取数据
- 2.入库:将爬取到的数据存入到数据库中
- 3.定时更新:通过定时任务定期爬取数据和更新数据
- 4.text-to-sql:用户输入一句话 将 function call 和 大语言模型.将这句话转为 sql
- 5.执行 sql: 执行 sql,从数据库中获取查询结果
- 6.对结果进行处理并展示给用户

## 结构化数据检索示例

- 1.用户输入: sjsu 2024 年秋季学期的一共有多少课程
- 2.text-to-sql:转换后的 sql 是 "select count(1) from 课程表"
- 3.执行 sql:执行结果 1024
- 4.展示给用户 1024
