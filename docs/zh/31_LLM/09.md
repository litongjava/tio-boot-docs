# 后置处理

## 什么是后置处理

后置处理事大模型的输出进行处理

## 为什么要进行 后置处理

提供更多和信息给用户

## 后置处理示例

- 1. 用户输入 "Math 241 的课程教授简介"
- 2. 通过文档检索,结构化检索,搜索数据,三方 API 等等获取到了教授的信息
- 3. 大模型输出了处理 后的教授信息,可能是长篇大论 1000 字
- 4. 对将 1000 字进行处理,比如提取成教授的名称
- 5. 提取到名称后从 教授名称表中进行检索
- 6. 对检索结果进行处理,因为检索后的书是结构化的数据非常好处理,例如生成一个教授名片
- 7. 将 教授名片 展示给用户
