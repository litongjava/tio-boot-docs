# 搜索数据

## 什么是搜索数据

搜索数据 是从搜索引擎上获取数据,将获取到的数据,用户的输入和提示器 作为输入,大模型输入推出

## 为什么要搜索数据

因为大模型的基础知识有限

## 搜索数据 示例

- 1. 用户输入 "SJSU 2024 年 秋季学期的开学时间是什么?"
- 2. 重写之后的输入 可能是 "SJSU 秋季 2024 开学时间"
- 3. 将重写之后的输入 输入到搜索引擎,返回 对应的数据 搜索结果片段
- 4. 对搜索结果片段 进行 重写之后的输入 进行相似度计算,找出较高相似度的片段
- 5. 将较高相似度的片段,用户的输入和意图识别提示词 最为一个输入交由大模型推理
