# 连接代码执行器

## 为什么要连接代码执行器

因为大模型不擅长 计算,但是大模型擅长编写代码,将大模型编写后的代码交给代码执行器

## 如何连接代码执行器

- 1. 意图识别: 判断用户的问题有计算类需求
- 2. 将 用户的输入,连接代码执行器 的 Funcation call 和 计算类提示词发送给大模型
- 3. 大模型编写代码 并返回
- 4. 连接代码执行器 执行代码
- 5. 对执行结果进程处理并展示给用户

## 连接代码执行器示例

- 1.用户输入 "檀香山 2023 年,有多少天气温高于 30 度"
- 2.通过三方 API 获取 2023 年的天气数据 共 365 条.这是结构化的数据
- 3.通过提示词模版 让大模型对 365 条结构化数据 编写处理代码
- 4.连接代码执行器 执行处理代码 结果是 12
- 5.对执行结果进程处理并展示给用户
