# 文档检索

## 什么是增强检索

增强检索的和相关资料一起分给哦大模型进行推理,主要的流程如下

- 1. 文档整理与格式化：首先，收集和整理知识，形成文档。注意文档中的大量文字直接用于 AI 学习可能不太利于处理，因此需要进一步的结构化或格式化。

- 2. 文档切片：将整个文档内容分割成多个段落，以便进行更细粒度的处理和检索。

- 3. 向量化：将分割后的文本段落转换为数字形式，即向量化，这些向量随后存储在向量数据库中。

- 4. 用户查询向量化：当用户提出问题时，同样将问题文本进行向量化处理，以便与文档中的内容进行匹配。

- 5. 向量预处理：对向量化后的用户查询进行必要的预处理，这可能包括规范化、去噪声等步骤。

- 6. 向量数据库检索：使用预处理后的查询向量在向量数据库中检索，寻找与用户问题相关性最高的知识段落。这些段落中可能包含了用户问题的答案。

- 7. 构造 Prompt 模版：将用户的问题和检索到的相关段落一同整合进一个预定义的 Prompt 模版中。

- 8. 大语言模型处理：将整合后的 Prompt 发送至大语言模型进行处理。

- 9. 模型生成答案：大语言模型根据输入的 Prompt 分析并生成回答，最后将答案返回给用户。

## 增强检索示例

- 1. 用户输入 "Math 241 课程的期末考试时间是什么"
- 2. 过问题重新,重写后的句子是可能是 "Math 241 期末考试时间"
- 3. 程序从向量数据和向 "Math 241 期末考试时间" 相似度较高的文档片段
- 4. 将 文档片段,用户输入和提示词 作为输入,大模型推理输出
