# 文档检索

## 什么是增强检索

增强检索是指通过结合大语言模型和相关资料进行推理，提升检索结果的准确性和相关性。主要流程如下：

1. **文档整理与格式化**：首先，收集和整理相关知识，形成文档。为了便于 AI 处理，需要对文档进行结构化或格式化。
2. **文档切片**：将文档内容分割成多个段落，以便进行更细粒度的处理和检索。
3. **向量化**：将分割后的文本段落转换为数字形式，即向量化，并将这些向量存储在向量数据库中。
4. **用户查询向量化**：当用户提出问题时，对问题文本进行向量化处理，以便与文档内容进行匹配。
5. **向量预处理**：对向量化后的用户查询进行必要的预处理，这可能包括规范化、去噪等步骤。
6. **向量数据库检索**：使用预处理后的查询向量在向量数据库中检索，寻找与用户问题相关性最高的知识段落。
7. **构造提示模板**：将用户的问题和检索到的相关段落整合进一个预定义的提示模板中。
8. **大语言模型处理**：将整合后的提示发送至大语言模型进行处理。
9. **模型生成答案**：大语言模型根据输入的提示分析并生成回答，最终将答案返回给用户。

## 增强检索示例

1. 用户输入："Math 241 课程的期末考试时间是什么？"
2. 经过问题重写，重写后的句子可能是："Math 241 期末考试时间"
3. 程序从向量数据库中检索与 "Math 241 期末考试时间" 相似度较高的文档片段
4. 将文档片段、用户输入和提示词作为输入，大语言模型进行推理输出答案

通过上述流程，增强检索能够有效地结合文档资料和大语言模型的能力，为用户提供准确且相关的回答。
