## 运行 manim

### 步骤 2：创建一个`modal_manim.py`脚本

```python
# modal_manim.py
import modal

stub = modal.Stub("manim-gpu-render")

# GPU镜像配置
image = modal.Image.debian_slim().pip_install(
    "manim[opengl]",
    "moderngl",
    "glcontext",
    "pyglet",
    "requests",
    "numpy",
    "moviepy"
)

@stub.function(
    image=image,
    gpu="T4",          # 可选GPU型号: T4、A100、A10G、V100
    timeout=600,       # 最大运行时间
)
def render_manim(scene_code: str):
    from manim import *

    # 临时文件夹（Modal环境提供）
    with open("/tmp/scene.py", "w") as f:
        f.write(scene_code)

    # 渲染Manim场景（OpenGL）
    import subprocess
    subprocess.run([
        "manim",
        "-ql",
        "--renderer=opengl",
        "/tmp/scene.py",
        "CombinedScene"
    ], check=True)

    # 返回渲染结果视频
    with open("/root/media/videos/scene/480p15/CombinedScene.mp4", "rb") as video_file:
        return video_file.read()

if __name__ == "__main__":
    with stub.run():
        scene_code = open("your_manim_scene.py").read()  # 本地Manim脚本
        video_content = render_manim.call(scene_code)
        with open("CombinedScene.mp4", "wb") as f:
            f.write(video_content)
        print("渲染完成，视频已下载。")
```

## 三、常见问题：

### Q：GPU 型号如何选择？

- **T4**：性价比最高，适合常规视频动画加速。
- **A100 或 V100**：更高性能，价格较贵，适合特别复杂的场景。

### Q：如何调整分辨率？

修改 Manim 的`-ql`(480p)、`-qm`(720p)、`-qh`(1080p)参数即可。

例如 1080p:

```bash
manim -qh --renderer=opengl scene.py CombinedScene
```
