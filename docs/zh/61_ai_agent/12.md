# 自建 youtube 字幕问答

youbue youtube 延迟非常高,本章介绍如何自建 youtube 视频问答

## 获取字幕

通过 supadata api 获取字幕

### supadata api

```
get https://api.supadata.ai/v1/youtube/transcript?videoId=kIo2BAubO6k
```

成功时的返回格式

```json
{
  "lang": "zh",
  "availableLangs": ["zh"],
  "content": [
    {
      "lang": "zh",
      "text": "去持有某一家公司",
      "offset": 1115600,
      "duration": 1266
    },
    {
      "lang": "zh",
      "text": "或者去人为的干预这个公司的决策",
      "offset": 1116866,
      "duration": 2200
    }
  ]
}
```
