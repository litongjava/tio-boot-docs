# HttpRequest

# 1. HttpRequest(Node remote)

  • 说明: 构造一个 HttpRequest 对象，并设置远程节点。
  • 用法:
    Node remoteNode = new Node("127.0.0.1", 8080);
HttpRequest httpRequest = new HttpRequest(remoteNode);

    2
